Fern is a toolkit for building REST APIs. With Fern, you can generate SDKs, API documentation, and server boilerplate.

Fern is compatible with OpenAPI, and supports TypeScript, Python, Java, Go, Ruby, C#, and PHP.

## Getting Started

Follow the [SDK Quickstart](https://buildwithfern.com/learn/sdks/overview/quickstart) to generate your first SDK, or the [Docs Quickstart](https://buildwithfern.com/learn/docs/getting-started/quickstart) to publish your first documentation site.
