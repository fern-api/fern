</dd>
</dl>

<dd>

**connection:** `Optional<String>` — Connection filter
    
</dd>
</dl>

<dl>
<dd>

**q:** `Optional<String>` — Query string following Lucene query string syntax
    
</dd>
</dl>

<dl>
<dd>

**searchEngine:** `Optional<String>` — Search engine version (v1, v2, or v3)
    
</dd>
</dl>

<dl>
<dd>
