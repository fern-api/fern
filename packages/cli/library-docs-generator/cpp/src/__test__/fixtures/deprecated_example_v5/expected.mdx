A [`strided_iterator`](/library/api/thrust::strided_iterator) wraps another iterator and moves it by a specified stride each time it is incremented or decremented.

```cpp showLineNumbers={false}
#include <thrust/strided_iterator.h>
```

<Error title="Deprecated">
Use `cuda::strided_iterator` instead.
</Error>

<AccordionGroup>
<Accordion title="Template parameters">

<ParamField path="RandomAccessIterator" type="typename">
A random access iterator.
</ParamField>

<ParamField path="StrideHolder" type="typename">
Either a [runtime_value](/library/api/thrust::runtime_value) or a [compile_time_value](/library/api/thrust::compile_time_value) specifying the stride.
</ParamField>

</Accordion>
</AccordionGroup>

**Inherits from:** [`thrust::iterator_adaptor< strided_iterator< RandomAccessIterator, StrideHolder >, RandomAccessIterator >`](/library/api/thrust::iterator_adaptor) (public), `StrideHolder` (private)
