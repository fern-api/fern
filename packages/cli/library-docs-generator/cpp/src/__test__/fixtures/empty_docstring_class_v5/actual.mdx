---
title: "cuda::buffer"
description: ""
---

`buffer` is a container that provides resizable typed storage allocated
from a given memory resource . It handles alignment,
release and growth of the allocation. The elements are initialized during
construction, which may require a kernel launch.

In addition to being type-safe, `buffer` also takes a set of
properties  to
ensure that e.g. execution space constraints are checked at compile time.
However, only stateless properties can be forwarded. To use a stateful
property, implement get_property(const buffer&, Property) .

```cpp showLineNumbers={false}
#include <cuda/buffer.h>
```

<Warning>
This constructor does *NOT* initialize any elements. It is the user's responsibility to ensure that the elements within `[vec.begin(), vec.end())` are properly initialized, e.g with `cuda::std::uninitialized_copy`. At the destruction of the `buffer` all elements in the range `[vec.begin(), vec.end())` will be destroyed.
</Warning>
