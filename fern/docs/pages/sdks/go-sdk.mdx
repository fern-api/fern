---
title: Fern | How to use Fern's Go SDK Generator
description: Use Fern's Go SDK generator and your API definition to output a fully functional Go SDK (client libraries) to your local filesystem, or publish it to GitHub. 
---

[Source code](https://github.com/fern-api/fern/tree/main/generators/go)<br />
**Latest version**: `0.12.0`

The Go SDK generator outputs a fully functional Go SDK.

## Local

Dump the generated SDK to the local file system. Available on the open source plan.

<CodeBlock title="generators.yml">
```yaml
groups:
  local:
    generators:
      - name: fernapi/fern-go-sdk
        version: 0.12.0
        output:
          location: local-file-system
          path: ../generated/sdk/go
```
</CodeBlock>

## Publish to GitHub 

Publish the generated SDK to GitHub.

<CodeBlock title="generators.yml">
```yaml
groups:
  publish:
    generators:
      - name: fernapi/fern-go-sdk
        version: 0.12.0
        github:
          repository: fern-imdb/imdb-go # replace imdb with your organization
```
</CodeBlock>
