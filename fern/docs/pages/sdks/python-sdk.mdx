---
title: Fern | How to use Fern's Python SDK Generator
description: Use Fern's Python SDK generator and your API definition to output a fully functional Python SDK (client libraries) to your local filesystem, or publish it to GitHub.
---

[Source code](https://github.com/fern-api/fern/tree/main/generators/python)<br />
**Latest version**: `0.8.0`

The Python SDK generator outputs a fully functional Python
SDK. It can publish the SDK to [the Python Package Index at PyPI.org](https://pypi.org) or to a private repository hosted by Fern.

## Local

Dump the generated SDK to the local file system. Available on the open source plan.

<CodeBlock title="generators.yml">
```yaml
groups:
  local:
    generators:
      - name: fernapi/fern-python-sdk
        version: 0.8.0
        output:
          location: local-file-system
          path: ../generated/sdk/python
```
</CodeBlock>

## Publish an internal package

Publish the generated SDK to a private PyPI repository hosted by Fern. Available on the Starter plan.

<CodeBlock title="generators.yml">
```yaml
groups:
  internal:
    generators:
      - name: fernapi/fern-python-sdk
        version: 0.8.0
        output:
          location: pypi.buildwithfern.com
          package-name: imdb # replace with your package name
```
</CodeBlock>

## Publish a public package

Publish the generated SDK to PyPI. Available on the Starter plan.

<CodeBlock title="generators.yml">
```yaml
groups:
  public:
    generators:
      - name: fernapi/fern-python-sdk
        version: 0.8.0
        output:
          location: pypi
          package-name: imdb
          token: ${PYPI_TOKEN}
        github:
          repository: imdb/imdb-python    
```
</CodeBlock>
