## 0.4.0
**`(feat):`** Introduces a `collection-name` configuration that allows you to control 
the name of the collection in the generated Postman collection. 

```yml
groups: 
  generators: 
    postman: 
      - name: fernapi/fern-postman
        version: 0.4.0
        config: 
          collection-name: My collection name
```
