## 0.11.12-rc1
**`(chore):`** - Support `x-fern-ignore` OpenAPI extension. This extensions configures fern to ignore certain endpoints when generating SDKs. (@dsinghvi) 
  ```yaml
  paths: 
    my/endpoint/path: 
      get: 
        x-fern-ignore: true # <------- fern will skip this endpoint
  ```
