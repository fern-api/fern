## 0.8.7
**`(feat):`** You can now specify publishing metadata to populate your POM on publish:
```yaml
generators:
  - name: fernapi/fern-java-sdk
    version: 0.X.Y
    output:
      location: maven
      registryUrl: ""
      publish-metadata:
        author: ""
        email: ""
        package-description: ""
        reference-url: ""
```
