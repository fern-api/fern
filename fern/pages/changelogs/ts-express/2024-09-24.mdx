## 0.17.5
**`(feat):`** Introduces a configuration `noOptionalProperties` which forces consumers to explicitly pass 
in `undefined` instead of excluding the key. To turn this setting on, do the following: 

```yml generators.yml
groups: 
  server: 
    generators: 
      - name: fernapi/fern-typescript-express
        version: 0.17.5
        config: 
          noOptionalProperties: true
```
